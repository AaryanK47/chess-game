# Pro Chess

A fully functional, modern Chess game built with vanilla HTML, CSS, and JavaScript.

## Features

- **Complete Chess Logic**: Supports all standard moves including Castling, En Passant, and Pawn Promotion.
- **Move Validation**: Prevents illegal moves and highlights valid options.
- **Game States**: Detects Check, Checkmate, and Stalemate.
- **Modern UI**: Dark theme with glassmorphism effects and smooth animations.
- **Responsive**: Playable on desktop and mobile devices.

## How to Run

Simply open `index.html` in any modern web browser. No build step or server required.

## Technologies

- HTML5
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (ES6+)
